Pasos para una Filtración Segura

  1. Prepara tu Evidencia

    Antes de enviar cualquier documento, foto o audio, elimina todos los metadatos (autor, ubicación, fecha de creación). Borra cualquier información personal que pueda identificarte. Usa herramientas como ExifTool o aplicaciones de limpieza de metadatos.

  2. Conéctate de Forma Segura

    Usa una conexión privada y confiable. Evita WiFi públicos, cibercafés o redes compartidas. Si es posible, utiliza una VPN confiable o la red Tor para ocultar tu ubicación real.

  3. Abre tu cuenta en proton mai

    Te recomendamos no abrir cuenta con tus datos reales, crea un alias o usa correos temporales. ProtonMail es nuestra eleccion de confianza

  4. Redacta tu Mensaje

    NUNCA incluyas tu nombre real, datos personales, o información que pueda identificarte. Describe los hechos de forma objetiva, incluye contexto relevante y especifica por qué es importante para el interés público.

  5. Envía y Mantente Seguro

    Después de enviar tu filtración, elimina cualquier rastro de la comunicación de tus dispositivos. Nuestro equipo se pondrá en contacto contigo solo si es absolutamente necesario para verificar información.

Cifrado PGP Avanzado

Para filtraciones extremadamente sensibles, recomendamos usar cifrado PGP. Este método garantiza que solo nosotros podamos leer tu mensaje.

Ejemplo de cómo usar PGP

Así es como puedes cifrar tu mensaje o archivo con nuestra clave pública: 

# 1) Guarda nuestra clave pública y añádela a tu llavero:
gpg --import clave-publica.asc

# 2) Escribe tu mensaje en un archivo de texto:
nano mensaje.txt

# 3) Cifra el archivo para Corvus News:
gpg --encrypt --armor --recipient corvusleaks@protonmail.com mensaje.txt

# Esto generará mensaje.txt.asc que puedes adjuntar en tu correo ProtonMail.

Si usas ProtonMail, también puedes activar PGP integrado directamente al redactar el correo: adjunta el archivo cifrado o usa la opción de cifrado avanzado desde la configuración de seguridad.

Consejos de Seguridad Adicionales

Usa Tor Browser

Para máximo anonimato, accede a nuestro sitio web a través de Tor Browser. Esto oculta tu dirección IP real.

Dispositivos Limpios

Si es posible, usa un dispositivo que no esté asociado contigo personalmente para realizar la filtración.

Elimina Metadatos

Siempre limpia los metadatos de fotos, documentos y archivos antes de enviarlos. Pueden contener información que te identifique.

Múltiples Copias

Mantén copias seguras de tu evidencia en múltiples ubicaciones por si necesitas enviar información adicional.

Tiempo de Espera

No esperes respuesta inmediata. Nuestro proceso de verificación puede tomar días o semanas dependiendo de la complejidad.

Comunicación Mínima

Solo nos comunicaremos contigo si es absolutamente necesario para verificar información. Nunca pediremos datos personales.
